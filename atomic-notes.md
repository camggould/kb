Atomic note taking is a great way to organize information to make it easily searchable.

The specific technique used by kb is Zettelkasten. Read more here: https://zettelkasten.de/introduction/

Create a knowledge base (kb) to store information. There is an associated mcp server for this tool, so just ask your favorite LLM to commit information to your kb. You can also ask it to check the kb for information you need to recall.
